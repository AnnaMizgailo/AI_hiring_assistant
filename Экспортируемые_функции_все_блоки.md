# Экспортируемые функции

Проект: AI-Ассистент для подбора персонала

---

## Блок 1 --- Веб / вакансии (`app.py`)

### Экспортируемые функции:

- `def register_routes(app): ...`\
  Подключает эндпойнты:
  - `/api/jobs`
  - `/api/jobs/{id}`
  - `/api/jobs/{id}/applications`

- `def create_job(db, recruiter_id, title, description, threshold_score, requirements_json=None) -> dict`\
  Возвращает:

  ```json
  { "job_id": "string" }
  ```

- `def list_jobs(db, recruiter_id) -> list[dict]`\
  Возвращает список вакансий рекрутера.

## Блок 2 --- Парсинг резюме (`parsing.py`)

### Экспортируемые функции:

- `def save_resume_file(store, candidate_id, file_bytes, filename, mime_type) -> dict`\
  Возвращает:

  ```json
  { "document_id": "string", "file_path": "string" }
  ```

- `def extract_text_from_file(file_path, mime_type) -> str`\
  Возвращает извлечённый текст.

- `def parse_resume_to_profile(raw_text) -> (profile_json, confidence_json, missing_fields_json)`\
  Возвращает структурированный профиль кандидата.

- `def upsert_candidate_profile(store, candidate_id, profile_json, confidence_json, missing_fields_json)`\
  Сохраняет профиль кандидата.

## Блок 3 --- RAG-скоринг (`scoring_rag.py`)

### Экспортируемые функции:

- `def index_job(store, job_id) -> int`\
  Индексирует вакансию.

- `def index_resume(store, document_id) -> int`\
  Индексирует резюме.

- `def retrieve_evidence(store, job_id, candidate_id, k=5) -> list[dict]`\
  Возвращает релевантные фрагменты.

- `def calculate_score(requirements_json, profile_json, evidence_chunks)`\
  Возвращает:

  ```json
  (score, rationale, missing_requirements, evidence_snippets)
  ```

- `def save_scoring(store, application_id, score, rationale, missing, evidence)`\
  Сохраняет результат скоринга.

- `def score_application(store, application_id) -> dict`\
  Возвращает:

  ```json
  {
    "application_id": "string",
    "score": 85,
    "rationale": "string",
    "missing_requirements": [],
    "evidence_snippets": []
  }
  ```

## Блок 5 --- Google Calendar + агент (`calendar_llm.py`)

### Экспортируемые функции:

- `def get_google_oauth_url(store, recruiter_id) -> str`\
  Возвращает ссылку авторизации.

- `def handle_google_callback(store, recruiter_id, code) -> dict`\
  Сохраняет токены.

- `def get_available_slots(store, job_id, recruiter_id) -> list[dict]`\
  Возвращает:

  ```json
  [
    {
      "slot_id": "string",
      "start_dt": "datetime",
      "end_dt": "datetime",
      "label": "string"
    }
  ]
  ```

- `def book_slot(store, application_id, slot_id) -> dict`\
  Возвращает:

  ```json
  {
    "status": "SCHEDULED",
    "event_id": "string",
    "start_dt": "datetime",
    "end_dt": "datetime"
  }
  ```

- `def generate_screening_summary(store, application_id) -> str`\
  Возвращает краткое summary.

- `def generate_feedback(store, application_id) -> str`\
  Возвращает персонализированный фидбек.
